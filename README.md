
# 📊 E-Commerce Sales Data Analysis Dashboard

This project presents an interactive **E-Commerce Sales Dashboard** built in **Power BI** to analyze sales performance, customer behavior, and business operations.

## 🎯 Objectives
- Analyze overall sales performance.
- Monitor key business metrics using KPIs.
- Identify top-performing products.
- Evaluate customer purchasing patterns.
- Examine payment method preferences.
- Track coupon code usage.
- Analyze referral sources.
- Understand order status distribution.

---

## 📊 Dashboard Highlights

### Key Performance Indicators (KPIs)
- **Customers:** 1.19K
- **Products:** 7
- **Regions:** 655
- **Orders:** 1.20K
- **Total Revenue:** $1.26M

### Visualizations
- Customer Coupon Code Status
- Order Status by Product
- Order Status Count
- Payment Method Distribution
- Total Revenue by Product
- Referral Source Analysis

---

## 🔍 Key Insights
- The business generated **$1.26M** in total revenue from over **1.20K orders**.
- **Chair** and **Printer** are the highest revenue-generating products.
- Customer purchases are fairly distributed across different coupon codes.
- Online payment is the most frequently used payment method.
- Orders are distributed across Delivered, Pending, Cancelled, Returned, and Shipped statuses, indicating opportunities to improve order fulfillment.
- Referral traffic comes from multiple channels including Instagram, Email, Google, Facebook, and direct referrals, showing diversified customer acquisition.
